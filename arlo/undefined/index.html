MPICHI3 并行计算环境搭建

发表于 2017-03-29 更新于 2017-04-21 分类于 HPC

新工作接触到HPC高性能计算项目，最近看了一些HPC高性能计算的理论资料，今天搭建MPICHI 并行计算环境，算是开始HPC的学习之路吧！

系统环境

CentOS 6_x64

设置hosts

vim /etc/hosts

新建用户

配置机器ssh无密码连接

安装MPICHI（三台服务器上执行以下相同操作）

获取mpichi

https://www.mpich.org/downloads/

安装配置MPICHI

$ vim ~/.bash_profile
$ export PATH=$PATH:/usr/local/mpich/bin/
$ source ~/.bash_profile
$ echo $PATH

新建machinefile

$ vim ~/machinefile
